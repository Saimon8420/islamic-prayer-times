Place your adhan MP3 files in this directory with these exact names:

- `adhan_makkah.mp3`    — Makkah adhan
- `adhan_madinah.mp3`   — Madinah adhan
- `adhan_alafasy.mp3`   — Mishary Alafasy adhan
- `adhan_abdulbasit.mp3` — Abdul Basit adhan

Each file should be a valid MP3 audio. Keep files under 1MB for best results.
You can find free adhan recordings online.

If a file is missing, the app will fall back to the device default notification sound.
